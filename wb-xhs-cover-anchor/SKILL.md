# 小红书封面锚点

先读取路由器交来的 Visual Brief。只使用已确认事实、已获授权的定性改写或中性默认值。

## 选择一个锚点方向

1. **可检视证据裁切**：有截图、原始物件或来源证据时，用可辨认局部承载承诺，不伪造证明。
2. **已授权主体锚点**：有可用人物、产品或真实主体授权时，让主体承担视觉焦点。
3. **已确认双状态对照**：有真实的前后两种状态时，用相邻对照表达变化；没有证据就不用结果对照。
4. **文字承诺锚点**：事实主要在标题里时，用大而短的精确标题和克制的支持物，避免把未知结果画成事实。

没有主体、证据或可确认对照时，使用**事实中性占位骨架**：大标题、抽象几何或中性物件、清晰留白，不暗示具体人物、业绩或案例。

## 交付步骤

1. 检查封面精确标题、画布和事实边界；标题未给时交回路由器预检。
2. 选一条锚点方向，写出“标题—视觉证据—不能声称什么”。
3. 生成或设计 1080×1440、3:4 画面；标题在手机缩略图仍可读。
4. QA：比例、标题逐字一致、层级、边距、对比度、事实边界。只修失败维度。
5. 持久化后回传真实路径和 QA 状态；未生成时明确 `not_called` 或 `not_run`。

## 边界

- 不把无来源的百分比、排名、收益或案例作为视觉事实。
- 不复制参考封面中的专属视觉资产。
- 不替代素材插画或黑白手绘专家。
