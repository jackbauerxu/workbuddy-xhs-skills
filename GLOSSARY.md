# Glossary

| Term | Meaning in this skill set |
|---|---|
| 小红书冷启动 | 从新号或低粉账号开始建立定位、内容、发布和复盘系统 |
| 变现倒推 | 先确定赚钱方式，再反推账号定位与内容方向 |
| 低粉爆款 | 粉丝不多但近期数据突出的内容样本 |
| 内容骨架 | 标题、开头、正文结构、标签和互动设计 |
| 账号档案 | 供 WorkBuddy 持续读取的账号定位、语气、边界和数据记录 |
| 选题库 | 按固定类型沉淀的标题公式和选题模板 |
| 去 AI 味 | 把初稿改成更像账号本人表达，并加入真实细节 |
| 排期复盘 | 把发布时间、内容类型和数据指标连接成下一轮决策依据 |
